CONTADOR DE PALAVRAS

Nessa aplicação desenvolvida em javascript é possível inserir um texto, frase, palavra e etc e logo abaixo saber quantos caracteres e palavras foram inseridos.

CALCULADORA DE PORCENTAGEM

De maneira muito rápida e fácil é possível calcular e obter o resultado da porcentagem de qualquer número inteiro.

RELÓGIO DIGITAL

SELETOR DE CORES

Deslize umas das barras para selecionar a cor que deseja.

As cores serão geradas em um código RGB no campo logo abaixo das barras deslizantes.

CONVERSOR DE TEMPERATURAS

Nesta aplicação é possível converter as 3 unidades de temperatura mais utilizadas, Celsius, Fahrenheit e Kelvin.

Digitando o valor em um dos campos de entrada e clicando em converter o usuário terá o valor das outras duas unidades logo abaixo de uma forma bem explicada.

Muito fácil de usar e com o resultado instantâneo após o click.

CALCULADORA IMC

Simples e fácil de usar, essa aplicação faz de forma rápida o cálculo do IMC (índice de massa corpórea) mostrando se você está no peso ideal para a sua altura.

CONVERSOR BINÁRIO / DECIMAL

Com apenas alguns cliques você pode converter qualquer número decimal inteiro para binário e de binário para decimal também.

Projeto bem simples com pouco javascript que cumpre com seu objetivo, com uma interface muito simplista facilita o uso para quem estiver acessando.

VISUALIZADOR DE BORDER RADIUS

Inspirado no repositório do github app-ideas específicamente no projeto Border-Radius-Previewer.

Atraves dessa aplicação o usuario vai digitar o valor em pixels de cada canto podendo alterar e visualizar o resultado do codigo em css.

PROJETOS UTILIZANDO O FRAMEWORK BOOTSTRAP

  • FASHION BLOG
  • START PAGE
  • MY BLOG
  • GOURMET CATERING

Para a aprendizagem de BOOTSTRAP resolvi usar os templates da página W3.CSS Templates como inspiração, são feitos em W3.CSS, um framework parecido com bootstrap, desenvolvi páginas com a maior semelhança possivel do original usando o bootstrap.

As páginas desenvolvidas em BOOTSTRAP são responsivas, isso quer dizer que elas se adaptam aos tamanhos de telas dos dispositivos trazendo uma melhor experiência ao usuário mas também um desenvolvimento e manutenção mais rápida para o desenvolvedor.

OBSERVAÇÃO: Quero deixar bem claro que eu não criei as páginas em bootstrap, apenas copiei a interface e não o código, não estou revindicando autoria sobre as páginas do w3c templates.
